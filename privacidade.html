Política de Privacidade

Última atualização: 17 de julho de 2026

Resumo: A sua privacidade é importante para nós. Recolhemos apenas os dados necessários para o funcionamento do serviço, não vendemos informações a terceiros e protegemos os seus dados com medidas de segurança adequadas.

1. Quem somos

O VMP SaaS é um serviço prestado por Adamgy Adamo Tembe, com base em Pemba, Moçambique. Somos responsáveis pelo tratamento dos seus dados pessoais no âmbito da utilização do nosso software e serviços.

2. Dados que recolhemos

Recolhemos apenas os dados estritamente necessários para prestar o nosso serviço:

2.1 Dados de identificação

  • Nome e email — para criar a sua conta e enviar comunicações sobre o serviço
  • Machine ID — identificador único do dispositivo para ativação da licença
  • Nome do negócio / cliente — para personalização do software e recibos

2.2 Dados de pagamento

  • Referências de pagamento — número de transação e valor (não armazenamos dados bancários)
  • Histórico de subscrições — plano contratado, datas de início e fim, estado da licença

2.3 Dados de utilização

  • Logs de erro — para diagnóstico e melhoria do software
  • Estatísticas anónimas — número de vendas, produtos registados (sem identificação pessoal)

2.4 Dados que NÃO recolhemos

  • Não recolhemos BI ou NUIT dos utilizadores finais
  • Não recolhemos dados de cartão de crédito (pagamentos processados por terceiros)
  • Não recolhemos dados de localização GPS
  • Não acedemos a ficheiros pessoais do seu computador

4. Partilha de dados

Não vendemos os seus dados. Partilhamos informações apenas com:

  • SendGrid — para envio de emails transacionais (ativação, renovação, suporte)
  • Render / Vercel — infraestrutura de alojamento dos servidores
  • Autoridades competentes — apenas quando legalmente obrigados

Todos os nossos parceiros estão sujeitos a acordos de confidencialidade e tratam os dados apenas conforme as nossas instruções.

5. Segurança dos dados

Implementamos as seguintes medidas para proteger os seus dados:

  • Encriptação em trânsito — todas as comunicações usam HTTPS/TLS
  • Encriptação em repouso — base de dados protegida com encriptação AES-256
  • Acesso restrito — apenas pessoal autorizado acede aos dados
  • Backups regulares — cópias de segurança diárias
  • Monitorização — deteção de acessos suspeitos em tempo real

6. Retenção e eliminação

Conservamos os seus dados apenas pelo tempo necessário:

  • Dados de conta: durante a vigência da subscrição + 1 ano após cancelamento
  • Dados de pagamento: 5 anos (obrigação fiscal)
  • Logs técnicos: 30 dias

Pode solicitar a eliminação antecipada dos seus dados contactando-nos. Note que alguns dados podem ser retidos por obrigação legal.

7. Os seus direitos

De acordo com a legislação moçambicana e princípios internacionais, tem direito a:

  • Aceder aos seus dados pessoais
  • Rectificar dados incorretos ou desactualizados
  • Solicitar a eliminação dos seus dados ("direito ao esquecimento")
  • Restringir o tratamento em determinadas circunstâncias
  • Receber os seus dados num formato estruturado (portabilidade)
  • Opor-se ao tratamento para fins de marketing

Para exercer estes direitos, contacte-nos pelos canais indicados na secção 9.

8. Cookies e tecnologias similares

O nosso site utiliza cookies essenciais para:

  • Manter a sua sessão de navegação
  • Recordar as suas preferências de idioma
  • Analisar tráfego anónimo para melhorar o site

Não utilizamos cookies de rastreamento de terceiros (Facebook Pixel, Google Analytics, etc.).

9. Alterações a esta política

Podemos actualizar esta política periodicamente. Alterações significativas serão comunicadas por email com 30 dias de antecedência. A data da última actualização está indicada no topo desta página.
