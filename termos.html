2. Objeto do Serviço

O VMP SaaS é um sistema de ponto de venda (POS) e gestão para máquinas de vending, que inclui:

  • Software desktop (Flutter) para operação das máquinas
  • Painel de administração web para gestão
  • Sistema de licenciamento e subscrição
  • Relatórios de vendas e stock
  • Suporte técnico conforme o plano contratado
